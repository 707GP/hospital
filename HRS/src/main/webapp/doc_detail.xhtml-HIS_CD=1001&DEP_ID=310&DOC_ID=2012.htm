李秋波

主任医师

广州市第一人民医院

医生简介：李秋波，男。毕业于第一军医大学，医学硕士。肿瘤放射治疗专业主任医师。广州市抗癌协会第二届理事会理事。从事肿瘤临床治疗23年，擅长放疗、化疗、生物靶向治疗、造血干细胞移植等综合治疗各类恶性肿瘤，对疑难病例的诊断和处理也积累了丰富的经验。尤其精通代表21世纪放疗主流技术的CT模拟定位、三维肿瘤治疗计划系统、三维适形精确放疗、调强放疗、X刀技术等技术。 05年主持广州市医药卫生科技项目《分子靶向药物协同三维适形放疗治疗晚期肺癌的临床研究》；07年主持广州市科技攻关计划项目《西妥昔单抗协同三维适形放疗治疗局部晚期鼻咽癌的研究》；参与省科委基金资助项目《造血干细胞移植中GVHD的免疫监测》。开展了《I、II期声门癌三维适形放疗的治疗方案比较》、《妇科恶性肿瘤术后盆腔盒式放射治疗临床观察》、《MRI检查对鼻咽癌三维适形放射治疗影响的探讨》、《难治性及复发性非霍奇金淋巴瘤放射治疗疗效的观察》4个专题临床研究。发表专业论文20余篇。获中国人民解放军科技进步二 等奖两项。 每周三、五上午肿瘤专家门诊。
