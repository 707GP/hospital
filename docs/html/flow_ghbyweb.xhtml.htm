网站预约挂号流程

第一步，登录，如果您是新用户，需先注册再登录。

第二步，找到您想预约的医生，可通过以下方式找医生

1)通过首页快速检索直接搜索医生

2)通过选择医院>选择科室>选择医生

3)通过选择疾病>筛选医生

第三步，查看医生的排班时间，选择合适的时段进行预约。

第四步，选择预约类型，为本人或为子女预约。

第五步，号源锁定成功。确认预约就诊信息，选择支付方式进行支付。

第六步，支付成功之后，可查看订单。

~预约平台订单支持跨支付渠道支付功能，目前支持的支付方式有：网银支付、微信支付、支付宝支付 三种支付方式。其中微信支付/支付宝支付，目前仅支持已开通微信支付商户/支付宝商户的医院，如果您所选择的医院未开通微信支付商户或支付宝商户，则该医院订单不支持微信支付/支付宝进行付款。
