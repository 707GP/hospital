•如何网上注册？
登录http://www.guahao.gov.cn/,广州统一预约挂号系统，用户（包括成人、儿童）凭真实姓名、身份证（ 港澳台籍用户提供《港澳居民来往内地通行证》 、《台湾居民来往大陆通行证》 ） 、手机号注册，完善个人资料即注册完成。
•健康卡登录预约挂号网站失败？
健康卡必须为实名卡才可以登录，请携带身份证与健康卡到医院自助终端进行绑定。
•网站注册时一定要填写真实的身份证信息吗？
为了给您提供更好的服务，避免黄牛党炒号等问题，我们在预约取号等多个环节都需要确认您的身份信息，因此请您填写真实的身份证信息，以保证您能顺利取号就诊。
•我没有在预约网站注册过，但用支付宝预约过了，为什么我登录不了预约网站？
用户登录支付宝APP后进入广州健康通支付宝服务窗口进行预约挂号。如果用户未曾在我方预约网站注册，如需要进入预约网站、广州健康通APP、广州健康通微信公众号进行预约，则需要在预约网站重新注册，完善资料后预约平台信息将自动与用户在支付宝的预约信息绑定。
