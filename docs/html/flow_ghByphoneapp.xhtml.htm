手机App预约挂号流程

一、通过网站下载广州健康通手机APP。

二、手机APP预约流程。

下载广州健康通APP→注册→登录→点击预约挂号/当天挂号→选择医院→选择医生→选择预约时间→选择号源时间段→提交订单→订单支付→完成预约。
*后付费的订单需到医院现场缴费

三、注册：点击登录界面人型图像进行注册，已注册用户可直接登录。

四、挂号类型：为 本人预约、为子女预约

五、手机APP订单支付模式：预付费、后付费

1)预付费订单，用户下订单后需要在30分钟内完成支付，超时未支付的订单系统则自动取消。

2)后付费订单，成功下订单后需在预约就诊当天，在预约时间前提前30分钟到医院指定窗口或院内自助终端缴费取号。

六、支付工具

预约平台订单支持跨支付渠道支付功能，目前支持的支付方式有：网银支付、微信支付、支付宝支付 三种支付方式。其中微信支付/支付宝支付，目前仅支持已开通微信支付商户/支付宝商户的医院，如果您所选择的医院未开通微信支付商户或支付宝商户，则该医院订单不支持微信支付/支付宝进行付款。

七、取号

预付费订单，在预约就诊当天，在预约时间前提前30分钟凭身份证（成人/儿童）、预约成功短信到医院指定窗口或院内自助终端取号。
（如预约9：00-10：00号源，则需要在8：30取号）

后付费订单，在预约就诊当天，在预约时间前提前30分钟凭身份证（成人/儿童）、预约成功短信到医院指定窗口或院内自助终端缴费取号。
（如预约10：00-11：00号源，则需要在9：30取号）
